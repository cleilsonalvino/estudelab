05. Tags de Formatação

O que são Tags de Formatação?

As tags de formatação em HTML são usadas para alterar a aparência do texto e estruturar o conteúdo de uma página web. Elas ajudam a enfatizar informações e a melhorar a legibilidade.

Tags Comuns de Formatação

Algumas das tags de formatação mais utilizadas incluem:

  • <b> - Define um texto em negrito, enfatizando sua importância.
  • <i> - Define um texto em itálico, geralmente utilizado para destacar palavras ou frases.
  • <strong> - Semelhante a <b>, mas também indica que o texto é de grande importância.
  • <em> - Semelhante a <i>, mas indica ênfase no texto, sugerindo que é importante.
  • <u> - Define um texto sublinhado.
  • <s> - Utilizada para indicar texto riscado ou que não é mais relevante.
  • <small> - Reduz o tamanho da fonte em relação ao texto normal.
