08. Tag Iframe

O que é a Tag Iframe?

A tag <iframe> (inline frame) é utilizada para incorporar outra página web dentro de uma página HTML. Isso permite que o conteúdo de uma fonte externa seja exibido diretamente em um espaço designado da sua página, sem que o usuário precise navegar para fora dela.

Características da Tag Iframe

Algumas características importantes da tag <iframe> incluem:

  • Facilidade de uso - Permite a integração rápida de conteúdo externo, como vídeos, mapas ou outras páginas.
