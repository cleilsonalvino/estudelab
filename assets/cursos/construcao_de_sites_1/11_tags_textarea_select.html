10. Tag Textarea

O que é a Tag Textarea?

A tag <textarea> é usada em HTML para criar um campo de entrada de texto que permite aos usuários inserir várias linhas de texto. É frequentemente utilizada em formulários para coletar comentários, mensagens ou qualquer entrada textual mais longa que não se encaixaria em um campo de entrada comum.

Características da Tag Textarea

Algumas características importantes da tag <textarea> incluem:

  • Multilinhas: Ao contrário do campo <input>, que é geralmente usado para uma única linha de entrada, a <textarea> permite várias linhas, o que é ideal para respostas longas.
  • Redimensionável: Por padrão, a maioria dos navegadores permite que os usuários redimensionem a área de texto, ajustando sua largura e altura conforme necessário.
  • Placeholder: A tag pode incluir um atributo placeholder que fornece uma dica sobre o que deve ser inserido, ajudando a guiar o usuário.
  • Limitação de caracteres: Você pode usar os atributos maxlength e minlength para controlar quantos caracteres o usuário pode inserir.

Atributos Comuns

A tag <textarea> possui vários atributos que permitem personalizar seu comportamento:

  • rows - Define o número de linhas visíveis na área de texto.
  • cols - Define o número de colunas visíveis na área de texto.
  • placeholder - Exibe uma dica dentro da área de texto quando não há texto inserido.
  • maxlength - Define o número máximo de caracteres que podem ser inseridos.
  • minlength - Define o número mínimo de caracteres que devem ser inseridos.
  • required - Indica que o campo deve ser preenchido antes do envio do formulário.
  • disabled - Desativa a área de texto, tornando-a não interativa.
  • select - Usado para criar uma lista suspensa de opções, permitindo que o usuário escolha uma entre várias opções disponíveis.
