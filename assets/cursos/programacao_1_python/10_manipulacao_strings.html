Manipulação de Strings

O que são Strings?

Strings são sequências de caracteres, como palavras ou frases. Em Python, as strings são representadas por texto entre aspas simples (') ou aspas duplas (").

texto = "Olá, mundo!"
print(texto)

Concatenando Strings

Podemos juntar (concatenar) duas ou mais strings usando o operador +.

nome = "Carlos"
saudacao = "Olá, " + nome + "!"
print(saudacao)

Neste exemplo, concatenamos as strings para criar uma saudação personalizada.

Interpolação de Strings

Uma maneira prática de inserir variáveis dentro de uma string é usando f-strings (formatação com f), disponíveis a partir do Python 3.6.

nome = "Maria"
idade = 30
mensagem = f"{nome} tem {idade} anos."
print(mensagem)

Aqui, usamos f"{variavel}" para interpolar as variáveis nome e idade na string.

Funções e Métodos Comuns para Strings

Python oferece vários métodos úteis para manipulação de strings:

  • len(): retorna o comprimento (número de caracteres) de uma string.
  • lower(): converte a string para letras minúsculas.
  • upper(): converte a string para letras maiúsculas.
  • strip(): remove espaços em branco no início e no final da string.
  • replace(): substitui partes da string por outra.
  • split(): divide a string em uma lista com base em um separador.

Exemplos

texto = "   Olá, Mundo!   "
print(len(texto))          # Saída: 15
print(texto.lower())       # Saída: "   olá, mundo!   "
print(texto.upper())       # Saída: "   OLÁ, MUNDO!   "
print(texto.strip())       # Saída: "Olá, Mundo!"
print(texto.replace("Mundo", "Python"))  # Saída: "   Olá, Python!   "
print(texto.split(","))    # Saída: ['   Olá', ' Mundo!   ']

Acessando Caracteres em uma String

Strings são indexadas, o que significa que podemos acessar caracteres individuais usando colchetes [] e um índice.

texto = "Python"
print(texto[0])  # Saída: P
print(texto[-1]) # Saída: n (índice negativo começa do final)

O índice 0 representa o primeiro caractere, e índices negativos começam do final da string.

Fatiamento de Strings

O fatiamento permite acessar uma sequência de caracteres dentro de uma string usando a notação [inicio:fim].

texto = "Programação"
print(texto[0:5])   # Saída: Progr
print(texto[6:])    # Saída: mação
print(texto[:5])    # Saída: Progr

Quando omitimos inicio ou fim, Python assume o início ou o final da string, respectivamente.

Exemplo Completo

Veja um exemplo que usa várias operações de manipulação de strings:

frase = " Aprender Python é divertido! "

# Remover espaços extras
frase = frase.strip()

# Converter para letras maiúsculas
print(frase.upper())

# Substituir uma palavra
nova_frase = frase.replace("divertido", "incrível")
print(nova_frase)

# Fatiar a string
print(frase[0:8])  # Saída: Aprender

# Dividir a frase em palavras
palavras = frase.split(" ")
print(palavras)  # Saída: ['Aprender', 'Python', 'é', 'divertido!']

Este exemplo demonstra como manipular uma string usando métodos comuns e fatiamento.
