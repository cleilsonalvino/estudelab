Função Input e Manipulação de Dados

O que é a Função input()?

A função input() é usada para capturar dados inseridos pelo usuário no console.

Ela permite que o programa interaja com o usuário, coletando informações para processamento.

Como Usar input()

Para utilizar input(), basta chamar a função e armazenar o resultado em uma variável.

Por padrão, o valor retornado por input() é sempre uma string (texto), então é necessário convertê-lo para outros tipos, se necessário.

nome = input("Qual é o seu nome? ")
print("Olá, " + nome)

Convertendo o Valor de input()

Se precisar de um número em vez de texto, você pode converter o valor de input() usando int() para inteiros ou float() para números decimais.

idade = input("Qual é a sua idade? ")
idade = int(idade)
print("Sua idade é:", idade)

Aqui, o valor inserido é convertido em um número inteiro usando int().

Exemplo: Somando Números com input()

Veja um exemplo onde pedimos ao usuário para inserir dois números e calculamos a soma:

num1 = input("Digite o primeiro número: ")
num2 = input("Digite o segundo número: ")

# Converte os valores para inteiros
num1 = int(num1)
num2 = int(num2)

soma = num1 + num2
print("A soma é:", soma)

Validando a Entrada do Usuário

É importante validar as entradas do usuário para garantir que os dados sejam do tipo esperado.

Podemos usar o método isdigit() para verificar se a entrada é um número antes de convertê-la.

num = input("Digite um número: ")

if num.isdigit():
    num = int(num)
    print("O número é:", num)
else:
    print("Por favor, digite um número válido.")

Neste exemplo, o programa verifica se a entrada é um número antes de realizar a conversão para evitar erros.

Exemplo Completo

Veja um exemplo completo usando input() para interagir com o usuário:

nome = input("Qual é o seu nome? ")
idade = input("Qual é a sua idade? ")

if idade.isdigit():
    idade = int(idade)
    print(f"Olá, {nome}! Você tem {idade} anos.")
else:
    print("Idade inválida! Por favor, insira um número.")

Este exemplo mostra como coletar informações do usuário e realizar verificações simples para garantir a integridade dos dados.
