Projeto Prático: Jogo de Adivinhação

Objetivo do Projeto

O objetivo deste projeto é criar um jogo de adivinhação em Python onde o jogador deve adivinhar um número escolhido aleatoriamente pelo programa.

Este projeto ajuda a praticar o uso de loops, condicionais, funções e geração de números aleatórios.

Estrutura do Jogo

O jogo será estruturado da seguinte forma:

  • O programa escolherá um número aleatório dentro de um intervalo definido (por exemplo, de 1 a 100).
  • O jogador terá várias tentativas para adivinhar o número.
  • O programa fornecerá dicas indicando se o palpite está muito alto ou muito baixo.

Passo 1: Importando o Módulo random

Para que o programa escolha um número aleatório, usaremos o módulo random do Python.

import random

Passo 2: Definindo a Função do Jogo

Vamos criar uma função chamada jogar_adivinhacao que executará a lógica do jogo.

def jogar_adivinhacao():
    numero_secreto = random.randint(1, 100)  # Escolhe um número aleatório entre 1 e 100
    tentativas = 0
    max_tentativas = 10

    print("Bem-vindo ao jogo de adivinhação!")
    print("Tente adivinhar o número entre 1 e 100.")

    while tentativas < max_tentativas:
        try:
            palpite = int(input("Digite seu palpite: "))
            tentativas += 1

            if palpite < numero_secreto:
                print("Muito baixo! Tente um número maior.")
            elif palpite > numero_secreto:
                print("Muito alto! Tente um número menor.")
            else:
                print(f"Parabéns! Você acertou o número em {tentativas} tentativas.")
                break
        except ValueError:
            print("Entrada inválida! Por favor, digite um número.")

    if tentativas == max_tentativas and palpite != numero_secreto:
        print(f"Você esgotou suas tentativas. O número secreto era {numero_secreto}.")

Passo 3: Configurando o Loop Principal do Jogo

Para permitir que o jogador jogue várias rodadas, vamos criar um loop que pergunta ao jogador se deseja continuar após cada rodada.

while True:
    jogar_adivinhacao()
    continuar = input("Deseja jogar novamente? (s/n): ")
    if continuar.lower() != 's':
        print("Obrigado por jogar! Até a próxima.")
        break
