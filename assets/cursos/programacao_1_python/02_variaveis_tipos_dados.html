Variáveis e Tipos de Dados

O que são Variáveis?

Variáveis são espaços na memória do computador onde armazenamos dados para uso em nossos programas.

Elas têm um nome e podem armazenar diferentes tipos de valores, como números, texto, listas, entre outros.

No Python, não é necessário declarar o tipo da variável explicitamente; o Python identifica automaticamente o tipo de dado atribuído.

Como Criar Variáveis em Python

Para criar uma variável, basta escolher um nome e atribuir um valor a ela usando o operador igual (=).

x = 5
nome = "Alice"
altura = 1.75

Aqui, x é uma variável inteira (int), nome é uma string (str) e altura é um número decimal (float).

Regras para Nomes de Variáveis

Os nomes de variáveis em Python devem seguir estas regras:

  • Devem começar com uma letra ou underline (_).
  • Não podem começar com números.
  • Podem conter letras, números e underscores.
  • Python diferencia letras maiúsculas e minúsculas (por exemplo, idade e Idade são variáveis diferentes).

Tipos de Dados em Python

Python possui diversos tipos de dados básicos. Os mais comuns são:

  • int - Números inteiros, como 1, 10 ou -3.
  • float - Números decimais, como 3.14 ou 0.5.
  • str - Sequências de caracteres (texto), como "Olá, mundo!".
  • bool - Valores booleanos, que podem ser True ou False.
  • list - Listas, usadas para armazenar vários valores em uma única variável, como [1, 2, 3].
  • dict - Dicionários, usados para armazenar pares de chave-valor, como {"nome": "Alice", "idade": 25}.

Exemplo de Uso de Variáveis e Tipos de Dados

Veja um exemplo simples usando diferentes tipos de dados e variáveis em Python:

nome = "Carlos"
idade = 28
altura = 1.82
ativo = True

print("Nome:", nome)
print("Idade:", idade)
print("Altura:", altura)
print("Ativo:", ativo)

Converter Tipos de Dados

Em Python, você pode converter tipos de dados usando funções como int(), float(), str(), etc.

x = "123"
y = int(x)  # converte string para inteiro
z = float(y)  # converte inteiro para float

A conversão de tipos é útil quando precisamos realizar operações específicas entre tipos diferentes.
