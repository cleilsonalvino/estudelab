Módulos e Bibliotecas

O que são Módulos?

Módulos são arquivos que contêm código Python e que podem ser importados para outros programas. Eles permitem reutilizar código em diferentes partes de um projeto.

No Python, um módulo pode conter funções, classes e variáveis que podem ser usadas em outros arquivos.

Importando Módulos

Para importar um módulo, usamos a palavra-chave import seguida do nome do módulo.

import math  # Importa o módulo de matemática

print(math.sqrt(16))  # Usa a função sqrt do módulo math para calcular a raiz quadrada

Neste exemplo, o módulo math é importado e a função sqrt (raiz quadrada) é utilizada.

Importando Funções ou Itens Específicos de um Módulo

Você também pode importar apenas uma função ou item específico de um módulo usando a palavra-chave from.

from math import sqrt

print(sqrt(25))  # Não é necessário usar math.sqrt, pois importamos diretamente sqrt

Aqui, apenas a função sqrt foi importada, permitindo chamá-la diretamente sem o prefixo math..

Bibliotecas Padrão do Python

O Python vem com uma biblioteca padrão, que inclui muitos módulos úteis para tarefas comuns, como manipulação de arquivos, operações matemáticas, manipulação de datas, e muito mais.

Alguns módulos populares da biblioteca padrão incluem:

  • math: funções matemáticas.
  • datetime: manipulação de datas e horários.
  • random: geração de números aleatórios.
  • os: operações com o sistema operacional.
  • sys: interação com o interpretador Python.

Exemplo de Uso de Módulos da Biblioteca Padrão

Veja alguns exemplos de como usar módulos da biblioteca padrão do Python:

import random

# Gera um número aleatório entre 1 e 10
numero_aleatorio = random.randint(1, 10)
print("Número aleatório:", numero_aleatorio)
from datetime import date

# Obtém a data atual
data_atual = date.today()
print("Data atual:", data_atual)

Instalando Bibliotecas Externas

Além da biblioteca padrão, você pode instalar bibliotecas externas para adicionar funcionalidades ao seu programa.

Para instalar uma biblioteca, você pode usar o pip, o gerenciador de pacotes do Python. Por exemplo, para instalar a biblioteca requests, que facilita fazer requisições HTTP:

pip install requests

Exemplo com uma Biblioteca Externa

Após instalar a biblioteca, você pode usá-la em seu código.

import requests

# Faz uma requisição HTTP para um site
resposta = requests.get("https://api.github.com")
print("Status da resposta:", resposta.status_code)

Neste exemplo, a biblioteca requests é usada para fazer uma requisição HTTP ao site do GitHub e exibir o código de status da resposta.

Exemplo Completo

Veja um exemplo completo que utiliza módulos da biblioteca padrão e uma biblioteca externa:

import math
import random
from datetime import date
import requests

# Usando o módulo math
raiz = math.sqrt(81)
print("Raiz quadrada de 81:", raiz)

# Usando o módulo random
numero = random.randint(1, 10)
print("Número aleatório entre 1 e 10:", numero)

# Usando o módulo datetime
hoje = date.today()
print("Data de hoje:", hoje)

# Usando a biblioteca requests
resposta = requests.get("https://api.github.com")
print("Status da requisição:", resposta.status_code)

Este exemplo demonstra como importar e utilizar módulos e bibliotecas para realizar várias operações.
