Funções

O que são Funções?

Funções são blocos de código que executam uma tarefa específica e podem ser reutilizados em diferentes partes de um programa.

Elas ajudam a organizar o código, tornando-o mais legível e fácil de manter.

Definindo uma Função

Para definir uma função em Python, usamos a palavra-chave def, seguida pelo nome da função e parênteses ().

O bloco de código da função deve estar indentado.

def saudacao():
    print("Olá, seja bem-vindo!")

Neste exemplo, criamos uma função chamada saudacao que exibe uma mensagem de boas-vindas.

Chamando uma Função

Para executar uma função, basta chamar o seu nome seguido de parênteses.

saudacao()  # Saída: Olá, seja bem-vindo!

Parâmetros e Argumentos

Funções podem receber dados de entrada chamados parâmetros. Esses dados são passados para a função quando ela é chamada.

def saudacao(nome):
    print(f"Olá, {nome}!")

Aqui, a função saudacao possui um parâmetro nome, que permite personalizar a saudação.

saudacao("Carlos")  # Saída: Olá, Carlos!

Valor de Retorno

Funções podem retornar um valor usando a palavra-chave return. Esse valor pode ser usado em outras partes do programa.

def soma(a, b):
    return a + b

Neste exemplo, a função soma retorna a soma dos dois parâmetros.

resultado = soma(3, 5)
print(resultado)  # Saída: 8

Funções com Múltiplos Parâmetros

Funções podem ter múltiplos parâmetros, permitindo o uso de vários valores como entrada.

def apresentar_pessoa(nome, idade):
    print(f"{nome} tem {idade} anos.")

Chamando a função com dois argumentos:

apresentar_pessoa("Ana", 25)  # Saída: Ana tem 25 anos.

Parâmetros Padrão

Você pode definir um valor padrão para um parâmetro. Esse valor será usado se nenhum argumento for passado para a função.

def saudacao(nome="visitante"):
    print(f"Olá, {nome}!")

Se você chamar a função sem passar um argumento, o valor padrão será utilizado.

saudacao()           # Saída: Olá, visitante!
saudacao("Maria")    # Saída: Olá, Maria!

Escopo de Variáveis

O escopo de uma variável define onde ela pode ser acessada no código.

No Python, variáveis definidas dentro de uma função têm um escopo local, ou seja, só podem ser usadas dentro da função onde foram definidas.

Exemplo de Escopo Local

def minha_funcao():
    mensagem = "Olá, mundo!"  # Variável local
    print(mensagem)

minha_funcao()  # Saída: Olá, mundo!
print(mensagem)  # Erro: mensagem não está definida fora da função

Neste exemplo, a variável mensagem só existe dentro da função minha_funcao.

Exemplo de Escopo Global

Variáveis definidas fora de qualquer função têm escopo global e podem ser acessadas em qualquer parte do código.

mensagem = "Olá, universo!"  # Variável global

def minha_funcao():
    print(mensagem)

minha_funcao()  # Saída: Olá, universo!
print(mensagem)  # Saída: Olá, universo!

Aqui, mensagem é uma variável global e pode ser acessada dentro e fora da função.

Exemplo Completo

Veja um exemplo que utiliza funções, parâmetros, retorno e escopo de variáveis:

def calcular_media(nota1, nota2, nota3):
    soma = nota1 + nota2 + nota3
    media = soma / 3
    return media

media_aluno = calcular_media(8, 7, 9)
print(f"A média do aluno é: {media_aluno}")

Neste exemplo, a função calcular_media calcula e retorna a média, e as variáveis dentro dela possuem escopo local.
