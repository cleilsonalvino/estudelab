Listas, Tuplas e Dicionários

O que são Listas?

Listas são estruturas de dados que armazenam múltiplos valores em uma única variável. Elas são mutáveis, o que significa que podem ser alteradas após a criação.

No Python, listas são definidas usando colchetes [], e os itens são separados por vírgulas.

frutas = ["maçã", "banana", "laranja"]
print(frutas)

Aqui, frutas é uma lista com três elementos: "maçã", "banana" e "laranja".

Operações com Listas

Algumas operações comuns com listas incluem:

  • append(): adiciona um item ao final da lista.
  • remove(): remove o primeiro item correspondente ao valor especificado.
  • len(): retorna o tamanho da lista.
  • index(): retorna o índice do primeiro item correspondente ao valor especificado.
frutas.append("uva")
print(frutas)  # ["maçã", "banana", "laranja", "uva"]

frutas.remove("banana")
print(frutas)  # ["maçã", "laranja", "uva"]

print(len(frutas))  # 3

O que são Tuplas?

Tuplas são semelhantes às listas, mas são imutáveis, o que significa que não podem ser alteradas após a criação.

Tuplas são definidas usando parênteses ().

cores = ("vermelho", "verde", "azul")
print(cores)

Aqui, cores é uma tupla com três elementos: "vermelho", "verde" e "azul".

Por que usar Tuplas?

Como tuplas são imutáveis, elas são mais rápidas que listas e são ideais para armazenar dados que não devem ser alterados.

O que são Dicionários?

Dicionários são estruturas de dados que armazenam pares de chave-valor. Cada chave é única e mapeia um valor.

Dicionários são definidos usando chaves {}.

aluno = {
    "nome": "Carlos",
    "idade": 20,
    "curso": "Engenharia"
}
print(aluno)

Aqui, aluno é um dicionário com três pares chave-valor: "nome" com o valor "Carlos", "idade" com o valor 20, e "curso" com o valor "Engenharia".

Operações com Dicionários

Algumas operações comuns com dicionários incluem:

  • get(): obtém o valor associado a uma chave.
  • keys(): retorna uma lista com todas as chaves do dicionário.
  • values(): retorna uma lista com todos os valores do dicionário.
  • items(): retorna uma lista de tuplas, cada uma contendo um par chave-valor.
print(aluno.get("nome"))  # Carlos
print(aluno.keys())         # dict_keys(['nome', 'idade', 'curso'])
print(aluno.values())       # dict_values(['Carlos', 20, 'Engenharia'])
print(aluno.items())        # dict_items([('nome', 'Carlos'), ('idade', 20), ('curso', 'Engenharia')])

Exemplo Completo

Veja um exemplo que usa listas, tuplas e dicionários:

# Lista de frutas
frutas = ["maçã", "banana", "laranja"]

# Tupla de cores
cores = ("vermelho", "verde", "azul")

# Dicionário de um aluno
aluno = {
    "nome": "Carlos",
    "idade": 20,
    "curso": "Engenharia"
}

print("Frutas:", frutas)
print("Cores:", cores)
print("Dados do aluno:", aluno)

Este exemplo demonstra como utilizar cada tipo de estrutura de dados para armazenar e manipular informações em Python.
