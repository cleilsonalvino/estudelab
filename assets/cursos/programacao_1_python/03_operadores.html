Operadores

O que são Operadores?

Operadores são símbolos especiais que realizam operações sobre valores e variáveis em Python.

Eles são usados para realizar cálculos matemáticos, comparar valores, e muito mais.

Tipos de Operadores em Python

Os principais tipos de operadores em Python incluem:

  • Operadores Aritméticos
  • Operadores de Atribuição
  • Operadores de Comparação
  • Operadores Lógicos
  • Operadores de Identidade
  • Operadores de Pertinência

Operadores Aritméticos

Usados para realizar operações matemáticas simples:

  • + (adição): soma dois valores (5 + 3 resulta em 8).
  • - (subtração): subtrai um valor de outro (5 - 3 resulta em 2).
  • * (multiplicação): multiplica dois valores (5 * 3 resulta em 15).
  • / (divisão): divide um valor por outro (5 / 2 resulta em 2.5).
  • // (divisão inteira): retorna o quociente inteiro da divisão (5 // 2 resulta em 2).
  • % (módulo): retorna o resto da divisão (5 % 2 resulta em 1).
  • ** (exponenciação): eleva um valor a uma potência (2 ** 3 resulta em 8).

Operadores de Atribuição

Usados para atribuir valores a variáveis:

  • = (atribuição simples): x = 5
  • += (atribuição com adição): x += 3 é o mesmo que x = x + 3
  • -= (atribuição com subtração): x -= 2 é o mesmo que x = x - 2
  • *= (atribuição com multiplicação): x *= 2 é o mesmo que x = x * 2
  • /= (atribuição com divisão): x /= 2 é o mesmo que x = x / 2

Operadores de Comparação

Usados para comparar valores e retornar um resultado booleano (True ou False):

  • ==: igual a (5 == 5 é True).
  • !=: diferente de (5 != 3 é True).
  • >: maior que (5 > 3 é True).
  • <: menor que (3 < 5 é True).
  • >=: maior ou igual a (5 >= 5 é True).
  • <=: menor ou igual a (3 <= 5 é True).

Operadores Lógicos

Usados para combinar expressões condicionais:

  • and: retorna True se ambas as condições forem verdadeiras ((5 > 3) and (3 < 4) é True).
  • or: retorna True se pelo menos uma das condições for verdadeira ((5 > 3) or (3 > 4) é True).
  • not: inverte o valor da condição (not (5 > 3) é False).

Operadores de Identidade e Pertinência

Outros operadores úteis incluem:

  • is e is not: verificam se duas variáveis apontam para o mesmo objeto.
  • in e not in: verificam se um valor está contido em uma sequência (como uma lista ou string).

Exemplos de Operadores

Veja alguns exemplos de uso de operadores em Python:

x = 10
y = 3

# Operadores aritméticos
print(x + y)   # Adição
print(x // y)  # Divisão inteira

# Operadores de comparação
print(x > y)   # Maior que

# Operadores lógicos
print((x > 5) and (y < 5))  # And lógico
