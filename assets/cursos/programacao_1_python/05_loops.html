Loops

O que são Loops?

Loops permitem que o programa execute um bloco de código repetidamente enquanto uma condição é verdadeira.

No Python, os loops mais comuns são for e while.

Loop for

O loop for é usado para iterar sobre uma sequência (como uma lista, tupla, string ou range).

for i in range(5):
    print(i)

Este exemplo imprime os números de 0 a 4. A função range(5) gera uma sequência de números de 0 até 4.

Looping em uma Lista

Você pode usar o for para iterar sobre os itens de uma lista.

frutas = ["maçã", "banana", "laranja"]

for fruta in frutas:
    print(fruta)

Loop while

O loop while repete um bloco de código enquanto uma condição é verdadeira.

contador = 0

while contador < 5:
    print(contador)
    contador += 1

Neste exemplo, o loop while continua enquanto contador for menor que 5, imprimindo o valor de contador e incrementando-o em 1 a cada iteração.

Palavras-chave break e continue

break e continue são palavras-chave que controlam o fluxo dos loops.

Usando break

A palavra-chave break interrompe o loop antes que ele complete todas as iterações.

for i in range(10):
    if i == 5:
        break
    print(i)

Este exemplo para o loop quando i é igual a 5.

Usando continue

A palavra-chave continue pula a iteração atual e vai para a próxima.

for i in range(5):
    if i == 3:
        continue
    print(i)

Neste exemplo, o número 3 é ignorado, e o loop continua com os próximos números.

Exemplo Completo

Veja um exemplo que utiliza loops for e while com break e continue:

numeros = [1, 2, 3, 4, 5]

for numero in numeros:
    if numero == 3:
        continue
    print(numero)

contador = 0
while contador < 5:
    if contador == 3:
        break
    print(contador)
    contador += 1

Este exemplo mostra como os loops podem ser usados para iterar e controlar o fluxo de execução com break e continue.
