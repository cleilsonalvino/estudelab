List Comprehension e Técnicas de Filtragem

O que é List Comprehension?

List comprehension é uma maneira concisa e eficiente de criar listas em Python. Com ela, podemos gerar listas com base em uma expressão, utilizando uma sintaxe mais enxuta.

Essa técnica facilita a criação de listas a partir de operações ou condições aplicadas a elementos de uma sequência.

Sintaxe Básica

A sintaxe básica do list comprehension é:

[expressão for item in sequência]

Por exemplo, para criar uma lista com os quadrados dos números de 1 a 5:

quadrados = [x**2 for x in range(1, 6)]
print(quadrados)  # Saída: [1, 4, 9, 16, 25]

List Comprehension com Condicionais

Podemos adicionar condições para incluir apenas certos elementos na lista. Isso é feito usando uma cláusula if ao final da expressão.

pares = [x for x in range(10) if x % 2 == 0]
print(pares)  # Saída: [0, 2, 4, 6, 8]

Neste exemplo, apenas os números pares de 0 a 9 são adicionados à lista pares.

List Comprehension com if e else

Também podemos incluir uma cláusula if e else dentro da expressão para aplicar diferentes operações com base em uma condição.

resultado = ["par" if x % 2 == 0 else "ímpar" for x in range(5)]
print(resultado)  # Saída: ['par', 'ímpar', 'par', 'ímpar', 'par']

Neste exemplo, a lista resultado contém "par" para números pares e "ímpar" para números ímpares.

Filtrando Listas com List Comprehension

List comprehension é útil para filtrar listas, permitindo-nos criar novas listas com base em condições específicas.

numeros = [1, 2, 3, 4, 5, 6]
maiores_que_tres = [x for x in numeros if x > 3]
print(maiores_que_tres)  # Saída: [4, 5, 6]

Aqui, apenas os números maiores que 3 são incluídos na lista maiores_que_tres.

Aninhando List Comprehension

Podemos aninhar list comprehension para criar listas de listas, útil para trabalhar com matrizes e combinações.

matriz = [[x * y for x in range(1, 4)] for y in range(1, 4)]
print(matriz)  # Saída: [[1, 2, 3], [2, 4, 6], [3, 6, 9]]

Neste exemplo, criamos uma matriz 3x3 com a tabuada dos números de 1 a 3.

Exemplo Completo

Veja um exemplo completo de list comprehension que aplica uma transformação e filtra uma lista:

numeros = range(10)
quadrados_pares = [x**2 for x in numeros if x % 2 == 0]
print(quadrados_pares)  # Saída: [0, 4, 16, 36, 64]

Aqui, calculamos o quadrado apenas dos números pares de 0 a 9.
