Estruturas Condicionais

O que são Estruturas Condicionais?

Estruturas condicionais permitem que o programa tome decisões com base em certas condições.

No Python, usamos a estrutura if, elif e else para executar blocos de código de acordo com condições específicas.

Sintaxe Básica

A estrutura condicional mais básica é o if. Caso a condição seja verdadeira, o bloco de código indentado será executado.

x = 10

if x > 5:
    print("x é maior que 5")

Usando else

O else define um bloco de código que será executado se a condição do if for falsa.

x = 3

if x > 5:
    print("x é maior que 5")
else:
    print("x é menor ou igual a 5")

Usando elif

O elif (abreviação de "else if") permite verificar múltiplas condições. Caso a primeira condição seja falsa, ele verifica a próxima.

x = 7

if x > 10:
    print("x é maior que 10")
elif x > 5:
    print("x está entre 6 e 10")
else:
    print("x é menor ou igual a 5")

Condicionais Aninhadas

É possível aninhar estruturas condicionais para verificar condições mais complexas.

x = 15
y = 20

if x > 10:
    if y > 15:
        print("x é maior que 10 e y é maior que 15")

Operadores Condicionais em Condições

Você pode usar operadores lógicos como and e or para combinar condições dentro de uma estrutura condicional.

x = 8
y = 3

if x > 5 and y < 5:
    print("x é maior que 5 e y é menor que 5")

Exemplo Completo

Veja um exemplo completo usando estruturas condicionais:

nota = 85

if nota >= 90:
    print("Aprovado com distinção!")
elif nota >= 70:
    print("Aprovado")
else:
    print("Reprovado")

Neste exemplo, diferentes mensagens são exibidas dependendo do valor da variável nota.
