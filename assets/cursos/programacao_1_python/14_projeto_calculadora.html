Projeto Prático: Calculadora

Objetivo do Projeto

O objetivo deste projeto é criar uma calculadora básica em Python que permita realizar operações matemáticas como adição, subtração, multiplicação e divisão.

Este projeto ajuda a aplicar conceitos de funções, condicionais e manipulação de erros, integrando-os em um programa funcional.

Estrutura da Calculadora

A calculadora será organizada da seguinte forma:

  • Funções para cada operação: adição, subtração, multiplicação e divisão.
  • Uma função principal para exibir o menu e capturar as escolhas do usuário.
  • Tratamento de erros para garantir que entradas inválidas não causem a interrupção do programa.

Passo 1: Definindo as Funções de Operação

Primeiro, criaremos funções para cada operação matemática. Cada função receberá dois parâmetros e retornará o resultado da operação.

def adicionar(a, b):
    return a + b

def subtrair(a, b):
    return a - b

def multiplicar(a, b):
    return a * b

def dividir(a, b):
    try:
        return a / b
    except ZeroDivisionError:
        print("Erro: Não é possível dividir por zero.")
        return None

Passo 2: Função Principal do Menu

Agora, criaremos uma função que exibe o menu de opções para o usuário e coleta suas escolhas. Essa função chamará a operação correspondente com base na escolha.

def calculadora():
    print("Selecione a operação:")
    print("1. Adição")
    print("2. Subtração")
    print("3. Multiplicação")
    print("4. Divisão")
    
    opcao = input("Digite a opção (1/2/3/4): ")

    if opcao in ['1', '2', '3', '4']:
        try:
            num1 = float(input("Digite o primeiro número: "))
            num2 = float(input("Digite o segundo número: "))

            if opcao == '1':
                print("Resultado:", adicionar(num1, num2))
            elif opcao == '2':
                print("Resultado:", subtrair(num1, num2))
            elif opcao == '3':
                print("Resultado:", multiplicar(num1, num2))
            elif opcao == '4':
                print("Resultado:", dividir(num1, num2))
        except ValueError:
            print("Erro: Entrada inválida. Por favor, insira números.")
    else:
        print("Opção inválida.")

Passo 3: Executando a Calculadora

Para que a calculadora funcione repetidamente, podemos usar um loop que pergunte ao usuário se deseja fazer outro cálculo.

while True:
    calculadora()
    continuar = input("Deseja fazer outro cálculo? (s/n): ")
    if continuar.lower() != 's':
        print("Encerrando a calculadora.")
        break
